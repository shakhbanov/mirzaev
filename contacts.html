Как добраться

ГКБ № 20 расположена на Коммунистическом проспекте в Советском районе Ростова-на-Дону. Удобный подъезд из любого района города и из ближайших городов Ростовской области.

  • На общественном транспорте: автобусом или маршруткой до остановки на Коммунистическом проспекте.
  • На автомобиле: у клиники есть парковка. Координаты для навигатора: 47.206696, 39.622235.
  • Из Батайска / Аксая: через мост по Будённовскому проспекту, далее по Коммунистическому. ~25–35 минут без пробок.
  • Из Новочеркасска / Таганрога: по трассе М-4 «Дон», далее по Коммунистическому проспекту. ~50–70 минут.
